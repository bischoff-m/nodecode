# Basics


## Node programs

A **node program** consists of nodes and connections. Each node can have input and output
sockets. Every input socket can have a connection that represents data flow into the node.

This sample program applies an operator to two numbers and outputs the result.

Each **node** represents a **function** that calculates output values from input values
and its state. The code below shows how this node could be defined in python (it's only
an example though).

```python
def operator2d(value1, value2):
    if(state.operator == '+')
        return value1 + value2
    if(state.operator == '-')
        return value1 - value2
    if(state.operator == '*')
        return value1 * value2
    if(state.operator == '/')
        return value1 / value2
    if(state.operator == '%')
        return value1 % value2
```

All nodes are built from UI components (like **labels, text fields, selects,
etc.**) that are predefined by nodecode.

The fields that make up a node and other information are defined in JSON format.

```json
{
    "id": "operator2d",
    "title": "Operator 2D",
    "fields": [
        {
            "id": "input-output-1",
            "type": "InputOutput",
            "props": {
                "inputLabel": "Value 1",
                "outputLabel": "Result"
            }
        },
        {
            "id": "input-output-2",
            "type": "InputOutput",
            "props": {
                "inputLabel": "Value 2"
            }
        },
        {
            "id": "operator",
            "type": "SegmentedControl",
            "props": {
                "options": ["+", "-", "*", "/", "%"]
            }
        }
    ]
}
```
